Woo 是一个本地优先的 Markdown 笔记桌面应用，基于 Electron、Vue 3 和 SQLite 构建。

## 开发

```bash
cd app-desktop
npm install
npm run dev          # Vite 开发服务器
npm run electron:dev # Electron 开发模式
```
