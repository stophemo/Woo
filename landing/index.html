为什么选择 Woo

融合现代编辑器体验与本地数据主权，让笔记回归纯粹

☁️

云端同步

基于 Supabase 的多设备实时同步，增量推拉 + 冲突检测。手机记灵感，电脑继续写，无缝衔接。

📝

所见即所得

基于 Tiptap (ProseMirror) 的富文本编辑器，支持 Markdown 语法输入，实时预览，流畅切换。

🤖

AI 辅助写作

集成 Google Gemini 与 DeepSeek，支持续写、摘要、翻译、头脑风暴。带来你的 API Key 即可使用。

⏱️

版本历史

自动保存 + 手动版本快照，随时回退到任意历史版本。重大编辑自动提交，绝不错过任何灵感。

📶

离线可用

所有数据本地留存，断网也不影响写作。网络恢复后自动同步，你的笔记始终完整。

🔒

隐私安全

开源透明，密码锁保护。数据经过你的 Supabase 项目传输，基础设施完全由你掌控。
