关于「无我」

「无我」源自佛教哲学，意为超越自我执念。Woo 以此命名，希望提供一个简约、专注的写作环境，让你放下对排版、格式、平台的焦虑，回归文字本身。 所有数据存储在本地 SQLite，下载即用；登录后可选 Supabase 同步，多设备互联。你的笔记，永远属于你。

无我 — 自由

写作不设限，让思想自由流动。没有模板束缚，没有格式焦虑，只有文字与思考。

自然 — 自主

数据在你手中，本地 SQLite 存储，无需依赖云端。你的笔记永远属于你。

专注 — 沉浸

沉浸式 Markdown 写作体验，心无旁骛。Tiptap 编辑器让写作如行云流水。

功能特性

v0.4.7 新增 AI 智能助手：自然语言操控笔记、语义搜索、流式编辑，实时展示思考过程

本地优先架构

所有数据存储在本地 SQLite 数据库。无需注册账号、无需后端服务、无需联网，下载即用。数据文件可直接复制备份或迁移，你拥有笔记的完全所有权。

所见即所得编辑

基于 Tiptap / ProseMirror 的富文本编辑器。支持标题、列表、代码块、引用等常用格式，实时 Markdown 渲染，书写与预览合一。

跨平台支持

基于 Electron 构建，适用于 Windows（7+）和 macOS（Apple Silicon）。一套数据，随处访问。Linux 支持开发中。

可选云同步

登录后开启 Supabase 云端同步，多设备数据互联。最后写入者胜出合并策略，确保数据一致性。云同步纯属可选，数据始终由你掌控。

版本历史记录

每次保存自动建立快照，24小时/7天分层合并策略。智能差异跟踪，支持回滚到任意历史版本，再也不怕误操作。

AI 智能助手

基于 Agent 工具调用框架的 AI 助手（支持 DeepSeek / Gemini / OpenAI）。语义搜索笔记、通过对话创建/修改/删除笔记、流式内容写入编辑器、实时展示思考过程。
