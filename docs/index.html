关于「无我」

「无我」源自佛教哲学，意为超越自我执念。Woo 以此命名，希望提供一个简约、专注的写作环境，让你放下对排版、格式、平台的焦虑，回归文字本身。

无我 — 自由

写作不设限，让思想自由流动。没有模板束缚，没有格式焦虑，只有文字与思考。

自然 — 自主

数据在你手中，本地 SQLite 存储，无需依赖云端。你的笔记永远属于你。

专注 — 沉浸

沉浸式 Markdown 写作体验，心无旁骛。Tiptap 编辑器让写作如行云流水。

功能特性

精心打造的开箱即用笔记体验，一切只为让你专注写作

本地优先架构

所有数据存储在本地 SQLite，你拥有笔记的完全所有权。无需注册账号，无需后端服务，下载即用。

所见即所得编辑

基于 Tiptap / ProseMirror 的富文本编辑器。实时 Markdown 渲染，书写与预览合一，排版随心。

跨平台支持

基于 Electron 构建，适用于 Windows、macOS、Linux。一套数据，随处访问，无缝切换工作流。

可选云同步

登录后即可开启 Supabase 云端同步，多设备数据互联。你的数据始终由你掌控，云同步纯属可选。

版本历史记录

智能差异跟踪的自动版本管理。重大更改自动建立快照，24小时/7天分层合并策略，可随时回滚。

暗色主题

精心设计的深色紫主题，降低长时间写作的眼部疲劳。两种主题自由切换，日夜写作同样舒适。
